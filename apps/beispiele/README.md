# Beispiele

Example integrations for Macheseinfach:

- **UI gallery** — `apps/website` imports `@macheseinfach/ui`
- **CLI** — `bun run --filter @macheseinfach/cli dev`
- **Secrets** — see root README (`env:decrypt` / `env:encrypt`)

Add subfolders here as patterns stabilize (e.g. `ocr-client/`, `webhook-handler/`).
